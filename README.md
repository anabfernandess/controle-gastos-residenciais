# Controle de Gastos Residenciais

Sistema para gerenciamento de pessoas, transações financeiras e consulta de totais residenciais.

## Tecnologias

### Back-end
- C#
- .NET
- ASP.NET Core Web API
- Entity Framework Core
- SQLite

### Front-end (Em desenvolvimento)
- React
- TypeScript

## Funcionalidades

- Cadastro, listagem e exclusão de pessoas;
- Cadastro e listagem de transações;
- Validação de receitas para menores de idade;
- Exclusão em cascata das transações ao remover uma pessoa;
- Consulta de receitas, despesas e saldos por pessoa;
- Consulta dos totais gerais da residência.
